SAÉ 3.02
C1 — Administer C3 — Programme
Developing communicating applications

Setting up a communication service with Python code. Coding a given number of clients and routers, with 1 storage server (SQL) to allow a company to communicate securely across a network while encrypting messages in transit. The client code lets the user choose the number of routers to traverse, encrypting the message in a chain (Tor-like network). At each router hop, one layer is decrypted, until the last router sends the message in plain text to the recipient.

What I learned

Coding a complex asymmetrical encryption tech, to transfer messages safely. Coding a decryption tech to decode the same messages, and code a server to stock all the keys for encryption/decryption.

Python MariaDB Cryptographie

SAÉ 3.03
C1 — Administer C2 — Connect C4 — Secure C5 — Monitor
Designing a multi-site network.

Implementation of an architecture for a multi-site company. Creation and deployment of the entire infrastructure, including IP addressing, protocols, and services based on the company's requirements outlined in the call for tenders. The architecture was divided into three main parts: the LAN section, covering the two remote sites; the Services section, covering the services requested in the call for tenders; and the ISP section, covering the network linking the two remote sites. This project was carried out in a group of three, with one person responsible for each section, while also producing detailed documentation for each part to help users understand what was implemented, how it was done, and the tests demonstrating that each section functions correctly.

What I learned

his was my most complete networking project so far. I designed a full multi-site infrastructure from scratch — LAN, inter-site connectivity and services — which made me think about IP addressing, routing protocols and security all at the same time. Working from a real call for tenders format taught me how to translate business needs into technical choices, which is a skill beyond just configuring equipment. I also had to write detailed documentation for my section, explaining what I did, why, and showing it worked — a good exercise in clarity and accountability.

GNS3 Windows Server Virtual Box VLAN / VRRP / OSPF BGP
